Image Analyser

Background

The image analysis application is a desktop tool designed to help users analyse images by counting the number of transparent and non-transparent pixels. It provides a simple and intuitive interface for users to upload an image file, perform the analysis, and view the results.


Using the Application:

  • Image Selection: Click the "Select image for analysis" button to select an image file from your local machine. A image preview will be seen once loaded.
  • Preview Selection: Click the "Show selected image" button to see your selected image and associated metadata.
  • Image Analysis: Once an image is selected, click "Analyse selected image". The application will analyse the image and display the results, including the total number of pixels, transparent pixels, non-transparent pixels and the percentage of non-transparent pixel density.
  • Result Display: The analysis results will be displayed in the application window, providing insights into the composition of the selected image.
  • Analysing another image: In the analyst results window, select "Analyse another image" and repeat the above steps.
  • Closing the Application: Close the application window to exit the program.
